# LibraryManagementAPI - .NET E-Challenge 2026

A simple **REST API** to manage books in a library, built with **ASP.NET Core**.  

---

## Features

- List all books  
- Get book by ID  
- Search books by title  
- Add, update, and delete books  

---

```bash
git clone https://github.com/fatimazahratag/LibraryManagementAPI-.NetEChallenge2026.git
